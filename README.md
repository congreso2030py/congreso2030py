Congreso2030py
=============

## Requirements

NodeJS > 10.0

## Setup

```
git clone https://github.com/politicalwatch/congreso2030py.es.git
cd congreso2030py
npm install
cp src/config/index.js.example src/config/index.js
```

Finally, edit *config/index.js* file with your specific values.


## Run

```
npm run serve
```
